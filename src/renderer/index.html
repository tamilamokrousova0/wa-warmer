    Настройки прогрева

    Пауза между диалогами, мин (от / до)
    Случайный интервал в минутах, который аккаунт выжидает после разговора, прежде чем вступить в следующий. У каждого аккаунта свой ритм — берётся случайное число между «от» и «до».
    Отлёжка после привязки, ч
    Сколько часов новый аккаунт просто «висит» онлайн и не участвует в прогреве. Свежие номера, которые сразу активничают, WhatsApp часто банит — отлёжка это снижает. 0 — отключить.
    Пауза после ре-логина, ч
    Пауза после переподключения аккаунта, чтобы WhatsApp снял ~6-часовой спам-лимит перед возвратом в прогрев: аккаунт снова «висит» онлайн эти часы, прежде чем продолжить.
    Дневной лимит / аккаунт
    Максимум исходящих сообщений с одного аккаунта за сутки. Достигнув лимита, аккаунт до следующего дня отдыхает. Внутри дня объём набирается плавно (без всплесков).
    Дней разгона (ramp-up)
    Дневной лимит растёт постепенно первые N дней у нового аккаунта (день 1 — маленький, дальше больше).
    Новый чат каждые N дней
    Число разных собеседников растёт постепенно: день 1 — 1 чат (аккаунты общаются парами), затем +1 собеседник каждые N дней. Связи между аккаунтами постоянные.
    Активность с — до (час)
    Окно работы по локальному времени компьютера. Вне окна прогрев спит (не пишет ночью).
    Дней прогрева / готов (ready)
    Через сколько дней аккаунт считается прогретым и выходит из ротации. Внутри-страновой прогрев идёт до дня N−1, затем финальный буст, готов на дне N+1.
    Картинки / Ссылки / Голосовые
    Какие типы контента иногда отправлять (берутся из папки контента). Выключенные — не шлются.
    Уникализация
    Добавляет в конец каждого сообщения невидимые символы, чтобы два одинаковых по виду сообщения были разными для WhatsApp (антиспам). Текст визуально не меняется.

    Группы по странам: аккаунты одной страны прогреваются парами внутри группы (нужно ≥2 номера), у каждой группы — свой прокси в стране номера. Смена прокси перезапускает движок группы.
