# WA Warmer — план идей и развития

Роадмап улучшений. Сгруппировано по темам, внутри — по приоритету. В скобках —
какие возможности GOWA это использует.

> **Реализовано в v1.3.0:** webhook-приёмник входящих (частично п.1), голосовые/стикеры/опросы
> (п.2), реакции/цитаты/«опечатка+правка» (п.3), вход по коду + массовое добавление (п.5, кроме
> шардинга), дашборд статистики по кнопке + экспорт CSV (п.7). Ниже — что ещё осталось/можно углубить.

## 1. Реализм прогрева (самое ценное для антибана)

- **Реальные ответы на входящие** — подключить webhook GOWA (`PATCH /devices/{id}/webhook`,
  HMAC-подпись) → локальный сервер в main-процессе получает входящие и отвечает
  по-настоящему: реакция, цитирование (`reply_message_id`), пауза «прочитал → печатает → ответил».
  Сейчас диалоги мы «разыгрываем» сами; webhook даст настоящую двустороннюю логику.
- **Разные типы контента** — GOWA умеет голосовые (`/send/audio` voice note), стикеры
  (`/send/sticker`), видео/GIF (`/send/video`), документы (`/send/file`), геолокацию
  (`/send/location`), контакты (`/send/contact`), опросы (`/send/poll`). Добавить их в ротацию —
  переписка выглядит живее, чем «текст + картинка».
- **Реакции и цитаты** — иногда реагировать эмодзи (`/message/{id}/reaction`) и отвечать
  цитатой на предыдущее сообщение вместо нового текста.
- **Опечатки и правки** — изредка отправлять с «опечаткой» и через пару секунд редактировать
  (`/message/{id}/update`, лимит 15 мин) — очень человеческое поведение.
- **Связные диалоги** — шаблоны «вопрос → ответ», темы разговора, чтобы реплики были
  логичны, а не случайны. Разная «близость» пар (кто-то общается чаще).

## 2. Групповой прогрев

- **Небольшие группы** — создавать группы из своих аккаунтов (`POST /group`,
  `/group/participants`) и вести переписку в них. Групповая активность — сильный сигнал
  «живого» аккаунта. До 500 групп на аккаунт.
- Периодически менять аватар/имя аккаунта (`POST /user/avatar`, `/user/pushname`) — прогрев профиля.

## 3. Онбординг многих аккаунтов (для 50–100 штук)

- **Вход по коду вместо QR** (`/app/login-with-code`) — вводишь номер, получаешь 8-значный
  код, вводишь в телефоне. Быстрее, чем сканировать 100 QR.
- **Массовое добавление** — очередь добавления, импорт названий/номеров из CSV.
- **Проверка номера** (`/user/check`) — до логина убедиться, что номер зарегистрирован в WhatsApp.
- **Шардинг по процессам** — если один процесс GOWA не тянет 100 аккаунтов (память/сокеты),
  автоматически поднимать несколько инстансов GOWA и распределять аккаунты между ними.

## 4. Надёжность и восстановление

- **Авто-reconnect** — при обрыве использовать `/devices/{id}/reconnect` вместо простого
  «отключился»; помечать бан отдельно от временного дисконнекта.
- **Сценарий после бана** — подсветка забаненного аккаунта, предложение перелогина/замены.
- **Health-мониторинг** — `/health`, потребление ресурсов; предупреждать при перегрузке.
- **Бэкап/восстановление** — экспорт/импорт папки `data/` (сессии + контент + настройки).

## 5. Статистика и наблюдаемость

- **Графики** — отправлено/принято по дням, кривая разгона, активность по часам.
- **Таймлайн аккаунта** — история событий (логин, диалоги, бан).
- **Экспорт логов** — писать журнал в файл с ротацией; экспорт статистики в CSV.

## 6. Управление контентом (расширение)

- **Редактор прямо в окне** — не только txt-файлы, но и правка/добавление из UI, превью картинок.
- **Категории и веса** — разные наборы фраз (утро/вечер, приветствия/болтовня) и настройка частот.
- **Проверка на уникальность** — не давать разным аккаунтам слать идентичный текст (антипаттерн).

## 7. Антибан-политики (пресеты)

- **Пресеты прогрева** — «осторожный / обычный / быстрый» с готовыми кривыми ramp-up, лимитами,
  паузами; выбор одним кликом.
- **Постепенный ввод** — новые аккаунты сначала только принимают/читают, потом начинают писать.
- **Тихие периоды** — случайные «выходные» и паузы у отдельных аккаунтов.
- **Прокси на аккаунт** (whatsmeow поддерживает) — если аккаунты не должны палиться одним IP.

## 8. Прочее (что ещё даёт GOWA)

- Отметка «прочитано»/«звёздочка», архивирование/закрепление чатов (`/chat/*`),
  исчезающие сообщения (`/chat/{jid}/disappearing`).
- Пересылка сообщений (`/message/{id}/forward`), удаление у всех (`/revoke`).
- Отклонение входящих звонков (`/call/reject`) — уже как авто-опция.

---

### Ближайшие кандидаты (рекомендация)

1. **Webhook + реальные ответы/реакции** — качественный скачок реализма.
2. **Голосовые/стикеры/опросы в ротации** — дёшево, заметно оживляет.
3. **Вход по коду + массовое добавление** — практично для 50–100 аккаунтов.
4. **Авто-reconnect и разделение «дисконнект vs бан»** — надёжность.
